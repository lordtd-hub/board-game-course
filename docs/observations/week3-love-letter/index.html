ข้อมูลเก็บเป็น draft ในเครื่องนี้เท่านั้น ไม่มีฐานข้อมูลและไม่ส่งออกจาก browser

ข้อมูลพื้นฐาน

การ์ดที่คุณจำได้ว่าเคยถือหรือเห็นชัด

รอบที่จำได้ที่สุด

ข้อมูลหลักที่คุณใช้ตัดสินใจคืออะไร

ทำไมข้อมูลนี้ถึงสำคัญ

คำพูด การ bluff และการอ่านคน

คำพูดของผู้เล่นในโต๊ะน่าเชื่อแค่ไหน

เหตุผลของคำตอบนี้

ถ้าเล่นอีก คุณจะทำอย่างไรไม่ให้คนอื่นจับทางง่าย
