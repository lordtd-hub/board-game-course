Turn Loop

หนึ่งตาเล่นพร้อมกันทั้งโต๊ะ

  1. ดูไพ่ในมือและเลือก 1 ใบ
  2. วางคว่ำพร้อมกัน
  3. เปิดพร้อมกัน แล้วทำ action
  4. ส่งไพ่ที่เหลือให้คนข้าง ๆ
Interactive: ตาหนึ่งทำอะไร?
กดปุ่มเพื่อดูจังหวะของ 1 turn

Card Anatomy

อ่านการ์ด 3 จุด

1. สีการ์ด

บอกหน้าที่หลัก เช่น เหลือง = เงิน/การค้า, แดง = ทหาร

2. Cost

มุมบนซ้ายบอกว่าต้องจ่าย resource/coin อะไร หรืออาจสร้างฟรีด้วย chain symbol

3. Effect

ตรงกลางบอกผลที่ได้ทันที หรือคะแนน/เงื่อนไขปลายเกม

สูตรพูดในห้อง: “ดูสี ดูค่าจ่าย ดูผล”

Payment

Resource ใช้แล้วไม่หมด

  • ทรัพยากรเป็นกำลังผลิต ใช้ได้ทุก turn
  • ซื้อจากซ้าย/ขวาได้ถ้าเป็น starting, brown, grey
  • จ่าย coins ให้เพื่อน แต่เพื่อนไม่เสีย resource
  • ห้ามสร้างอาคารชื่อซ้ำ

Three Actions

ไพ่ 1 ใบ ทำได้ 3 ทาง

สร้างอาคาร

จ่าย cost แล้ววางการ์ดเปิดหน้า ได้ effect ตามการ์ด

สร้าง Wonder stage

จ่าย cost ของ stage แล้วคว่ำไพ่ใต้ Wonder เอฟเฟกต์ของไพ่ไม่ทำงาน

ขายไพ่

ทิ้งไพ่เพื่อรับ 3 coins ใช้เมื่อสร้างไม่ได้หรือไม่อยากให้คนอื่นได้ไพ่ใบนั้น

Wonder Board

Wonder คือแผนระยะกลางของเรา

  • แต่ละ stage มี cost และ reward ของตัวเอง
  • สร้างตามลำดับซ้ายไปขวา
  • ใช้ไพ่ใบใดก็ได้คว่ำเป็น stage
  • เกมแรกให้ใช้ Wonder เป็น “แผนสำรอง” เมื่อไพ่ในมือไม่เข้ากับเมืองเรา
ตัวอย่าง Wonder stage

จ่าย wood → ได้ 3 VP

จ่าย resource → ได้ effect

ได้เงิน/คะแนนท้ายเกม

Military

แดงไม่ได้ตีทั้งโต๊ะ เทียบแค่ซ้ายและขวา

  • นับท้ายแต่ละ Age
  • เทียบจำนวนโล่กับเพื่อนซ้ายและขวาแยกกัน
  • ชนะได้ victory token แพ้ได้ defeat token
  • เสมอไม่มีอะไรเกิดขึ้น

Guided First Age

กิจกรรมสอนเล่น 25 นาทีแรก

Turn 1-2

หยุดทุกโต๊ะ ให้ทุกคนพูดว่า “ฉันสร้าง/ขาย/สร้าง Wonder เพราะอะไร”

Turn 3-4

Teaching Script

สคริปต์สั้นสำหรับอาจารย์

เปิดคาบ

“เกมนี้ชนะจากการสร้างเมือง ไม่ใช่ทำทุกอย่างพร้อมกัน เลือกทางที่เมืองเราถนัด”

ตอนเจอสัญลักษณ์ใหม่

“ดูสีของการ์ดก่อน แล้วถามว่า effect นี้เกิดทันทีหรือปลายเกม”

ตอนผู้เล่นงง resource

“resource เป็นโรงงาน ใช้แล้วไม่หาย แต่ 1 symbol ใช้ได้ครั้งเดียวต่อ turn”

ตอนจบ Age I

“ตอนนี้ทุกคนเล่นเป็นแล้ว ที่เหลือคือเลือกแผนให้ชัดขึ้น”
