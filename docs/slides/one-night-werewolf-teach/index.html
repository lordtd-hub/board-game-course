One Night Werewolf

คืนเดียว ตื่นมาเถียงกัน แล้วโหวตให้ถูกคน

สไลด์สำหรับคนที่ยังไม่เคยเล่น: จัดโต๊ะ รู้หน้าที่ของบทบาท เล่นรอบแรก และตัดสินทีมชนะ

เป้าหมาย

ดูจบแล้วควรเริ่มเล่นรอบแรกได้เลย

  • จัดโต๊ะได้ถูก: ผู้เล่นแต่ละคนมีการ์ด 1 ใบ และมีการ์ดกลาง 3 ใบ
  • ตามลำดับกลางคืนของชุดบทบาทเริ่มต้นได้
  • จำได้ว่าบางบทบาททำให้การ์ดของผู้เล่นเปลี่ยนได้
  • โหวตและตัดสินทีมชนะได้ โดยไม่ต้องเปิดคู่มือกติกากลางเกม

จำไว้ก่อนเริ่ม

การ์ดที่คุณเห็นตอนเริ่มคืน อาจไม่ใช่การ์ดใบเดิมตอนเช้า เพราะ Robber, Troublemaker และ Drunk อาจสลับการ์ดบนโต๊ะ

จัดโต๊ะ

จัดโต๊ะ: จำนวนการ์ด = จำนวนผู้เล่น + 3

  • สับการ์ดบทบาท แล้วแจกให้ผู้เล่นคนละ 1 ใบ
  • วางการ์ดที่เหลือ 3 ใบคว่ำกลางโต๊ะ เรียกว่า “การ์ดกลาง”
  • ทุกคนดูการ์ดตัวเอง แล้วคว่ำไว้ที่เดิม อย่าขยับตำแหน่งมั่ว
  • ใช้แอปหรือผู้บรรยายอ่านลำดับกลางคืนให้ทุกคนทำตาม

ลำดับรอบ

หนึ่งรอบมี 5 ช่วง จำตามนี้ก็เล่นได้

1

แจกการ์ด

ทุกคนดูบทบาทเริ่มต้นของตัวเอง

2

กลางคืน

บางบทบาทตื่นมาดูหรือสลับการ์ดตามลำดับ

3

คุยกัน

ทุกคนอ้างบทบาท ถามกัน และจับพิรุธ

4

โหวต

ชี้พร้อมกัน เลือกคนที่น่าจะเป็น Werewolf

5

เปิดไพ่

เปิดการ์ดตอนจบ แล้วตัดสินว่าทีมไหนชนะ

คำที่ต้องย้ำ: บทบาทที่เห็นตอนเริ่มคืน อาจไม่ใช่บทบาทของคุณตอนจบคืน

ชนะอย่างไร

จบเกมด้วยการโหวตครั้งเดียว

ทีม Village ชนะ

ถ้าโหวตทำให้ Werewolf อย่างน้อย 1 คนตาย

ทีม Werewolf ชนะ

ถ้ามี Werewolf อยู่ในวง และไม่มี Werewolf คนใดตาย

กรณีไม่มี Werewolf ในวง

ทีม Village ควรโหวตกระจายให้ทุกคนได้ไม่เกิน 1 เสียง เพื่อไม่ให้ใครตาย

Tanner เป็นบทบาทเสริม

ถ้าใส่ Tanner เขามีเป้าหมายพิเศษ: อยากถูกโหวตตาย

บทบาทเริ่มต้น

เริ่มจากบทบาทเหล่านี้ก่อน โต๊ะจะไม่ซับซ้อนเกินไป

กดชื่อบทบาทเพื่อดูว่าแต่ละตัวตื่นตอนไหน และทำอะไร

ตัวอย่างรอบกลางคืน

ตัวอย่าง: การ์ดเปลี่ยนตำแหน่งอย่างไรในคืนเดียว

เริ่มคืน

ทุกคนรู้การ์ดเริ่มต้นของตัวเอง แต่ยังไม่มีใครรู้ว่าบทบาทจะถูกสลับหรือไม่

จุดสังเกต: ความจริงบนโต๊ะกับความเชื่อในหัวผู้เล่นยังตรงกันอยู่

กระดานคำอ้าง

เวลาคุย ให้ถามว่า “ที่พูดมารู้จากบทบาทไหน?”

คำอ้าง A

“ผมเป็น Seer ดูการ์ดกลาง เห็น Villager กับ Drunk”

คำอ้าง B

“ฉันเป็น Troublemaker สลับ A กับ C”

คำอ้าง C

“ผมเป็น Villager ไม่ได้เห็นการ์ดใคร แต่คิดว่า A พูดเร็วเกินไป”

เลือกคำอ้าง เพื่อดูว่าควรถามอะไรต่อก่อนโหวต

กติกาโหวต

โหวตพร้อมกัน แล้วคนที่ได้คะแนนสูงสุดตาย

กรณีปกติ

A ได้ 3 เสียง, B ได้ 1 เสียง

A ตาย

ถ้า A เป็น Werewolf ฝั่ง Village ชนะ

กรณีเสมอสูงสุด

A ได้ 2 เสียง, C ได้ 2 เสียง

A และ C ตาย

ถ้าในนี้มี Werewolf อย่างน้อย 1 คน Village ชนะ

กรณีกระจาย

ทุกคนได้คนละ 1 เสียง

ไม่มีใครตาย

ใช้ได้เมื่อคิดว่า Werewolf อยู่กลางโต๊ะ

เช็กก่อนเล่น

เช็กความพร้อมก่อนเล่นจริง

1. Troublemaker สลับ A กับ B แล้ว A รู้ตัวไหม?

2. Drunk สลับกับการ์ดกลาง แล้วดูการ์ดใหม่ไหม?

3. Werewolf คนเดียวทำอะไรเพิ่มได้?

4. Seer เลือกดูอะไรได้?

กดคำตอบเพื่อดูเฉลยสั้น ๆ

สคริปต์อาจารย์

สคริปต์สั้นสำหรับรอบสอน

  • “ทุกคนดูบทบาทของตัวเอง แล้วคว่ำไว้ที่เดิม”
  • “ทำ action ตอนกลางคืนเงียบ ๆ ตามลำดับ ห้ามส่งสัญญาณ”
  • “ตอนเช้า ทุกคนพูดได้ทุกอย่าง แต่ให้ฟังว่าอะไรเป็นหลักฐาน อะไรเป็นแค่คำอ้าง”
  • “โหวตพร้อมกัน อย่าเปลี่ยนมือหลังเห็นคนอื่น”

ถ้าห้องยังงง

เล่นรอบแรกแบบเปิดไพ่สอน 1 รอบก่อน แล้วค่อยเล่นจริงแบบปิดไพ่ รอบแรกมีไว้ให้เห็นว่าการ์ดถูกสลับอย่างไร ไม่ต้องสนว่าใครชนะมากนัก

พร้อมเล่น

กติกาสั้นที่อยากให้นักศึกษาพูดซ้ำได้

1. ดูบทบาทเริ่มต้น

แต่บทบาทตอนจบอาจเปลี่ยนแล้ว

2. Night action เปลี่ยนเกม

บางคนเห็นการ์ด บางคนสลับการ์ด บางคนไม่รู้ว่าตัวเองเป็นอะไร

3. ตอนคุยต้องถามให้ชัด

ฟังคำอ้าง แล้วถามว่ารู้จากบทบาทไหน

4. โหวตคือคำตอบสุดท้าย

ถ้าโหวตโดน Werewolf ฝั่ง Village ชนะ

อ้างอิง

อ้างอิงกติกาและหมายเหตุสื่อ

สไลด์นี้ใช้ภาพการ์ดจำลองเพื่อการสอน ไม่ใช้ artwork จากเกมจริง และออกแบบให้เริ่มเล่น base game ได้ในห้องเรียน
