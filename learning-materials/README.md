# Learning Materials

พื้นที่นี้แยกออกจาก source code ของเว็บแอป เพื่อเก็บเนื้อหารายวิชาและเอกสารประกอบการสอนโดยเฉพาะ

## โครงสร้าง

- `bookref/` หนังสือและไฟล์อ้างอิงเดิมของรายวิชา
- `weekly-modules/` สำหรับสื่อรายสัปดาห์ในอนาคต
- `worksheets/` สำหรับ worksheet หรือ template ที่ใช้ในระบบ
- `rubrics/` สำหรับ rubric และเกณฑ์การประเมิน

## หลักการใช้

- เก็บไฟล์ต้นฉบับการสอนในนี้ ไม่ปนกับ `app/`, `lib/`, `components/`
- เมื่อจะนำเนื้อหาเข้าเว็บ ให้แปลง/คัดลอกเฉพาะข้อมูลที่ต้องใช้เข้าสู่ Google Sheet หรือระบบหน้าเว็บ
- ไฟล์ส่งงานของนักศึกษาไม่เก็บในโฟลเดอร์นี้ ให้เก็บใน Google Drive ตามโครงสร้างของระบบ
